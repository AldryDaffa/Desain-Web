Tujuan matakuliah ini adalah memahamkan tentang dasar HTML-script sebagai materi dasar membuat tampilan webpage sebelum dikenalkan berbagai aplikasi tampilan page yg lain

Tag 'meta' mendefinisikan metadata tentang dokumen HTML. Metadata adalah data (informasi) tentang data.
Meta description adalah sebuah cuplikan ataupun rangkuman konten website yang muncul di halaman hasil pencarian.
Meta keywords ini berisi informasi kata kunci pada sebuah website.
